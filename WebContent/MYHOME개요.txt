-- 회원 관리 기능

-- 댓글형 게시판 table  --> (공지사항 : notice, 댓글 :  reply )
    1. 게시글 테이블과 댓글 테이블이 별도로 존재한다.
    2. 외래키를 이용해 관계를 설정한다.
    	
    3. 페이징 처리한다. (MVC 패턴에서 사용, 페이징 처리 메소드)
    4. 게시글에 조회수를 처리한다.
    5. 게시글 삭제는 댓글 존재 유무에 따라 달려져야한다.
    	(댓글이 달린 게시글은 지울 수 없으므로)
    6. 로그인 한 회원만 작성할 수 있다.
    7. 검색 기능이 가능하다.
    
    
    

-- 계층형 게시판 (자유 게시판)
	1. 게시판 1개로 게시글과 댓글을 모두 저장한다.
	2. 게시글/댓글 구분을 위한 정보가 필요하다.
	3. 게시글/댓글 정렬을 위한 정보가 필요하다.
	4. 삭제하지 않는 삭제 기능을 사용한다.
	5. 페이징 처리한다. (page 객체를 이용한 초기 버전, 직접 모두 만들기.)
	
-- 이미지 게시판
	1. 댓글 처리 
	2. 삭제 시 ajax로 댓글 확인 후 삭제 진행..
	3. 댓글 리스트 ajax
